🧩 표준 템플릿 빠른 추가

8개 페이브 표준 모듈을 옵션과 함께 선택하면 SKU 리스트에 자동 추가됩니다. 다중 추가 가능 — 컬럼별로 패턴을 다르게 조합해 주문 구성을 만드세요.

패턴/옵션을 선택하면 추가될 SKU 미리보기가 여기에 표시됩니다.

1️⃣ SKU 리스트 또는 상품옵션 ID 붙여넣기

🆕 상품옵션 ID(8자리)도 지원! 옵션ID 붙여넣으면 매핑된 SKU 세트로 자동 펼침 + 옵션 1개 = 컬럼 1개로 자동 묶음 처리됩니다.
· 옵션ID 8자리 (예: 29504814) → 그 옵션의 SKU 자동 펼침
· SKU 9자리 (예: 100111575) → 기존대로 처리
· 수량 컬럼(옆 셀의 1~99 숫자) 자동 반영 · 둘 다 섞어도 OK
